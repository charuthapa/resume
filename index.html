NETWORK DEVICES:

REPEATER

• The main purpose of repeater is to receive a signal and then retransmits those signal.
• It is used to extend the transmission so that the signal can cover longer distance.
• It is a two port device.
• It operates at physical layer.
• It do not amplify the signal, when the signal becomes weak, it copy the signal bit by bit and then regenerates it at its original strength.
• It allows forwarding means it forwards the signal.
• It provides no filtering means it doesn’t filter signals.
• Collosion occurs and traffic is high because it’s a pure hardware device and doesn’t contain any software, it just forward the signals.
• It is not used in modern network.

HUB

• A network device that allows you to connect multiple computer to a single network.
• It is basically a multi-port repeater.
• It operates at physical layer.
• Generally, it is used to connect computers in a LAN.
• It is used to extend the transmission so that the signal can cover a longer distance.
• It provides the feature of broadcasting. Suppose, there are four devices i.e. A, B, C and D connected with hub, now device A wants to send some data to device C, then the hub forward that data to device C as well as all the devices connected with hub. This is because there is no software in hub which can provides filtering.
• Collosion occurs, the maximum number of collosion depends on the ‘n’ number of devices.
• There are three types of hub i.e. Passive hub, Active hub and Intelligent hub.
• It is not used in modern network.

SWITCH

• A switch is a multiport bridge device with a buffer and a design that can boost its efficiency and performance.
• It can perform error checking before forwarding data. This makes it more efficient and improves its performance as it forwards the good and efficient packet to the correct port only which doesn’t have errors.
• It operates at layer 2 of the OSI model that is the data link layer.
• It supports unicast, multicast and broadcast communications.
• It can easily identify that which device is connected with which port by using MAC address, that’s why it sends data packets on selective destination port.
• It is more intelligent or a better version of hub.

BRIDGE

• A bridge is a repeater with add on the functionality of filtering content by reading the MAC address of the source and the destination.
• It is a network device that is used for interconnecting two LAN’s working on same protocol.
• It operates at physical layer as well as data link layer.
• It provides filtering as it doesn’t forward data to all the devices but sends data to the device which you want to sent. It is done because of the data link layer.
• It reduces the amount of traffic on a LAN by dividing it into two segments.
• Collosion can be reduced easily.
• It is more intelligent as it first stores the MAC address of the destination and then broadcast data accordingly.
• There are three types of bridges i.e. Local bridge, Remote bridge and Wireless bridge.
